This page shows a sample of use og Gridster, a library to create dynamic grids in your application.

The code uses the library at http://gridster.net/.

The code is client side and follows the demo documentation in the site. First we create a structure of 'div' and 'li' to define the grid.

The next step is to initialize it via javascript. With that we have a functional grid in our page..
