This page shows a sample of use of Forms in HTML 5.

You can find more information at http://diveintohtml5.info/forms.html.

The code displays a form which contains:

  • Autofocus on the first element (fallback options are not shown, check http://diveintohtml5.info/forms.html for a list of possible fallbacks)
  • An input text with placeholder text
  • All the semantic HTML 5 elements, which will be rendered using browser capabilities (that is, if the browser supports them, you'll see a special render)

On submit the HTML 5 validations will automatically be triggered (try to put a non-valid URL or Email, for example)

Be aware that some elements, like 'number' or 'range' provide (if supported) Javascript methods to obtain the values directly as numbers (no need to cast from String) or tu modify the value via code.
