This page shows I18N values used in Javascript. The values are obtained from the 'messages' file in Play, allowing you to have 1 unique file a a source of internationalized strings.

The code uses the module at https://github.com/julienrf/play-jsmessages.

In the controller (object I18N) we define a method (jsMessages) that will return a javascript function. The content of this function is generated by the module above and will translate the keys to the proper I18N chains, as defined in our 'messages' file.

Then we embed a call to that function in a 'script' tag, which will let us use that function (Messages in this example) to obtain the I18N text.
