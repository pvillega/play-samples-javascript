This page shows how to obtain routes to be used in Javascript from teh routes defined in your 'routes' file.

It uses no modules.

The controller (object Routing) has a method (javascriptRoutes) that stores in the 'jsRoutes' object the paths defined.

At this point you can access that object to send requests to the generated paths. If the url requires parameters, we can provide them:

jsRoutes.controllers.Demos.listDemos(page = 0, orderBy = 1, nameFilter = '', versionFilter = -1, tagFilter = [value]).url
